# Youtup

 This code is written in python for the purpose of downloading youtube videos in mp3 format


## Installing

1. clone the code

```
git clone https://github.com/HoussemCharf/youtup.git
```

2. execute & run
```
python script.py
```
3. provide the youtube link

4. enjoy


## Requirements

* Cgitb lib
* Pytube lib
* Pydub lib
